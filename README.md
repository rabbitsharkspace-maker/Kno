# Kno — AI Knowledge Platform

Turn information overload into lasting knowledge.

Kno is a spatial knowledge OS powered by Gemini AI. 
Capture content from any source, build your personal 
knowledge graph, and surface insights you'd otherwise forget.

## Try it now
→ knoko.space (no install needed — bring your own Gemini API key)

## What it does
- Upload PDFs, paste URLs, or drop in voice notes
- AI automatically summarises and builds connections 
  between your ideas
- Logic Guard audits sources for factual accuracy
- Memory Lab tracks what you're retaining over time
- Infinite canvas to arrange your knowledge visually

## Why we built it
Most note-taking apps help you store information. 
Kno helps you actually understand and remember it.

## Tech stack

Supabase · D3.js · Framer Motion
